2.关系运算：

| 运算符 |   意义   |
| :----: | :------: |
|   ==   |   相等   |
|  ！=   |  不相等  |
|   >    |   大于   |
|   >=   | 大于等于 |
|   <    |   小于   |
|   <=   | 小于等于 |

由于计算机计算浮点数有误差，所以比较两个double类型数据可以把他们相减看是否小于一定限度内。
